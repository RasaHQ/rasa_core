rasa_core.actions.action module¶

class rasa_core.actions.action.Action[source]¶

Bases: object

Next action to be taken in response to a dialogue state.

name()[source]¶

Unique identifier of this simple action.

run(dispatcher, tracker, domain)[source]¶

Execute the side effects of this action.

Args:
tracker (DialogueStateTracker): the state tracker for the current user.
You can access slot values using tracker.get_slot(slot_name) and the most recent user message is tracker.latest_message.text.
dispatcher (Dispatcher): the dispatcher which is used to send messages back
to the user. Use dipatcher.utter_message() or any other Dispatcher method.

domain (Domain): the bot’s domain

Returns:
List: A list of Event instances
